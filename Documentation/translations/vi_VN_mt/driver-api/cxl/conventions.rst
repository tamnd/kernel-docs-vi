Tính toán Express Link: Quy ước Linux
#######################################

Tồn tại các nền tảng vận chuyển làm cong hoặc phá vỡ thông số kỹ thuật CXL
mong đợi. Ghi lại các chi tiết và lý do cho những sai lệch đó.
Mượn định dạng mẫu ACPI Code First để nắm bắt các giả định
và sự cân bằng sao cho việc triển khai nhiều nền tảng có thể tuân theo
cùng một quy ước.
