==
rv
==
--------------------
Xác minh thời gian chạy
--------------------

:Phần hướng dẫn sử dụng: 1

SYNOPSIS
========

ZZ0000ZZ ZZ0001ZZ [ZZ0002ZZ]

DESCRIPTION
===========

Xác minh thời gian chạy (ZZ0000ZZ) là một phương pháp nhẹ (nhưng nghiêm ngặt)
để xác minh chính thức với cách tiếp cận thực tế cho các hệ thống phức tạp.
Thay vì dựa vào mô hình chi tiết của một hệ thống (ví dụ:
thực hiện lại mức hướng dẫn), RV hoạt động bằng cách phân tích dấu vết
về việc thực hiện thực tế của hệ thống, so sánh nó với một quy trình chính thức
đặc tả hành vi của hệ thống.

Công cụ ZZ0000ZZ cung cấp giao diện cho tập hợp thời gian chạy
màn hình xác minh (rv).

COMMANDS
========

ZZ0000ZZ

Liệt kê tất cả các màn hình có sẵn.

ZZ0000ZZ

Chạy màn hình.

OPTIONS
=======

ZZ0000ZZ, ZZ0001ZZ

Hiển thị văn bản trợ giúp.

Đối với các tùy chọn khác, hãy xem trang man để biết lệnh tương ứng.
