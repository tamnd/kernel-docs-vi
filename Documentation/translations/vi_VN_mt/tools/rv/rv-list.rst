============
danh sách rv
============
---------------------------
Liệt kê các màn hình có sẵn
---------------------------

:Phần hướng dẫn sử dụng: 1

SYNOPSIS
========

ZZ0000ZZ [ZZ0001ZZ]

DESCRIPTION
===========

Lệnh ZZ0000ZZ in tất cả các màn hình có sẵn. Những màn hình này
có thể được kích hoạt bằng lệnh ZZ0001ZZ.

OPTIONS
=======

ZZ0000ZZ, ZZ0001ZZ

In menu trợ giúp.

SEE ALSO
========

ZZ0000ZZ\(1), ZZ0001ZZ\(1)

Tài liệu ZZ0000ZZ của nhân Linux:
<ZZ0001ZZ
