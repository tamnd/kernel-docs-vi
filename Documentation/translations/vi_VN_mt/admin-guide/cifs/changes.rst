Thay đổi
=======

Xem ZZ0000ZZ để biết tóm tắt
thông tin về các bản sửa lỗi/cải tiến đối với hỗ trợ CIFS/SMB2/SMB3 (các thay đổi
sang mô-đun cifs.ko) theo phiên bản kernel (và phiên bản mô-đun nội bộ cifs).
Điều này có thể dễ đọc hơn là phân tích đầu ra của
"git log fs/smb/client" theo bản phát hành.
