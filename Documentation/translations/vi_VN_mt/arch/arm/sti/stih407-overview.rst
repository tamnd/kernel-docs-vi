====================
Tổng quan về STiH407
====================

Giới thiệu
------------

STiH407 là SoC thế hệ mới dành cho set-top box Multi-HD, AVC
    và ứng dụng máy chủ/máy khách được kết nối cho vệ tinh, cáp, mặt đất
    và thị trường IP-STB.

Tính năng
    - ARM Cortex-A9 lõi kép 1,5 GHz CPU (28nm)
    - SATA2, USB 3.0, PCIe, Gbit Ethernet
