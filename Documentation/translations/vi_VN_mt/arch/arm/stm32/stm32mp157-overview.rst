Tổng quan về STM32MP157
=======================

Giới thiệu
------------

STM32MP157 là Cortex-A MPU nhắm đến nhiều ứng dụng khác nhau.
Nó có tính năng:

- Lõi ứng dụng Cortex-A7 lõi kép
- Bố cục hình ảnh 2D/3D với GPU
- Hỗ trợ giao diện bộ nhớ tiêu chuẩn
- Kết nối chuẩn, kế thừa rộng rãi từ dòng STM32 MCU
- Hỗ trợ bảo mật toàn diện
