======
libbpf
======

Nếu bạn đang muốn phát triển các ứng dụng BPF bằng thư viện libbpf thì đây
thư mục chứa tài liệu quan trọng mà bạn nên đọc.

Để bắt đầu, bạn nên bắt đầu với tài liệu ZZ0000ZZ, tài liệu này cung cấp hiểu biết ở mức độ cao về
API libbpf và cách sử dụng chúng. Điều này sẽ cung cấp cho bạn một nền tảng vững chắc để bắt đầu
khám phá và sử dụng các tính năng khác nhau của libbpf để phát triển BPF của bạn
ứng dụng.
